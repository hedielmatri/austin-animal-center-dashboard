# Austin Animal Center: Data Analysis & Dashboard

## About the Project
This project analyzes intake and outcome data from the Austin Animal Center (AAC). It was created as a personal data science project to explore animal shelter trends and build an interactive dashboard for exploring the data.

## Live Links
You can interact with the final products here:
* **[Interactive Shiny Dashboard](https://hedielmatri.shinyapps.io/Project3byHediELMATRI/)**: A web application allowing users to filter and visualize shelter data dynamically.
* **[Exploratory Data Analysis Report](https://rpubs.com/Dedushka/AASDA)**: A comprehensive RPubs report detailing the statistical exploration, data cleaning, and visualizations.

## Tools and Methods
* **Language:** R
* **Data Manipulation & Visualization:** Tidyverse, ggplot2
* **Web Deployment:** R Shiny
* **Reporting:** R Markdown (deployed to RPubs)

## Repository Structure
* `app.R`: The source code for the interactive Shiny dashboard.
* `analysis.Rmd`: The R Markdown file used to generate the EDA report.
